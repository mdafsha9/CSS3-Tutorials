Generic font families

CSS has five generic font families

  • Serif--serif fonts have strokes on each edge of a letter. They create a sense of formality and elegance
  • Sans-serif--these fonts have clean lines (no strokes attached on edges). They create a modern and minimalistic look.
  • Monospace--here all the letters have same fixed width. They create a meachanical look
  • Cursive--It imitates human hand writing.
  • Fantasy--it is decorative/playful fonts
