Transition Effects

CSS Transition effect allows we to change property value smoothly, over a given duration.
CSS Transition properties:

  • transition
  • transition-delay
  • transition-duration
  • transition property
  • transition-timing-function

How to use Transition?

  • the CSS property which we want to add an effect to
  • the duration of the effect
