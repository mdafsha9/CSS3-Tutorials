CSS Images

Image can improve design and appearance of the web page.

Responsive images

Responsive imagesare images to scale nicely to fit any browser size

widht set to 100%

in css property width is set to 100%, the image will be responsive and scale up and down.

max-width set to 100%

In CSS property max-width is set to 100%, and height is auto then the image scale to down and never scale up than the origin image size

Rounded images

border-radius property is used to create a rounded images
