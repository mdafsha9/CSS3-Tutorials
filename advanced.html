CSS box-sizing property:

The CSS box-sizing property allows us to include padding and border in an element's total width and height.

Before adding box-sizing property Example:

width 300px, height 100px and border

width 300px, height 100px and border. Here we added padding extra

After adding box-sizing property Example:

width 300px, height 100px and border

width 300px, height 100px and border. Here we added padding extra and box-sizing
