CSS Animation

CSS allows animation of HTML elements without JavaScript or Flash.
CSS animation properties are:

  • @keyframes
  • animation-name
  • animation-delay
  • animation-duration
  • animation-iteration-count
  • animation-direction
  • animation-timing-function
  • animation-fill-mode
  • animation

What are CSS animations ?

  • An animation let an element gradually change from one style to another.
  • we can change as many CSS properties as we want, as many times as we want
  • To use CSS animations, we must first specifie @keyframe for the animation
  • keyframe holds what style the elemeent will have at certain time

The @keyframe rule

  • When we specify CSS style inside @keyframe rule, the animation gradually changes from the current style to new style
  • to get an animation to work, we must bind the animation to an element

Run animation in Reverse direction or alternate cycles

The animation-direction property specifies whether an animation should be displayed in forward, backword or alternate cycles.

animation-direction properties are:

reverse
The animation should be played in reverse direction(reverse)
alternate
The animation should be played in forward direction first then backward direction
alternate-reverse
The animation should be played in backwards fisrt then forwards direction
