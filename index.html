CSS

Introduction

  • CSS is used to specify the layout of web pages
  • CSS describes how HTML elements should be displayed
  • CSS saves lot of time. It changes layout of multiple web pages all at once
  • External style sheets are stored in .css file

CSS Syntax

CSS rule set consist of selector and declaration block.

  • A selector points to HTML element which we want to style.
  • The declaration block contains one or more declarations, seperated by semi-colon.
  • Each declaration includes CSS property name and value, seperated by colon.
  • Multiple CSS declarations are seperated by semi-colon, and declaration block surrounded by curly braces.

CSS Selectors

CSS element selector:

The element selector selects HTML element based on the element name

CSS id selector:

id selector uses id attribute of an HTML element to select specific eleemnt

CSS class selector

The class selector uses class attribute of an HTML element to a specific element

Universal selector

Universal selector is used to select all the HTML elements.

Grouping selectors

Grouping selector is used to select group of element at a time

How to add CSS

There are three ways to add CSS3

External files
With an external file, it is used to change the look of the entire website by changing just one file.
Internal file
Internal file, uses style element inside the head section
Inline style
Inlie style, uses style attribute to the relevant element

CSS Background

CSS background properties is used to add background effects for HTML elements

  • background-color
  • background-image
  • background-repeat
  • background-attachment
  • background-position

CSS Borders

CSS border property allow we to specify the width, style and color of an element's border.

  • border-style
  • border-width
  • border-color
  • border
  • border-radius

All HTML elements consider as a box. In CSS, the term box-model is used for layout or design. In CSS box-model is essentially a box that wraps around every HTML element. It consists as a margin, border, padding and a content.

content
The content of the box. Where the text and images appear.
padding
It clears an area around the content. Padding is transparent.
border
border goes around the padding and content
margin
Clears an area outside the border. Margin is transparent.
