CSS

Introduction

  • CSS is used to specify the layout of web pages
  • CSS describes how HTML elements should be displayed
  • CSS saves lot of time. It changes layout of multiple web pages all at once
  • External style sheets are stored in .css file

CSS Syntax

CSS rule set consist of selector and declaration block.

  • A selector points to HTML element which we want to style.
  • The declaration block contains one or more declarations, seperated by semi-colon.
  • Each declaration includes CSS property name and value, seperated by colon.
  • Multiple CSS declarations are seperated by semi-colon, and declaration block surrounded by curly braces.

All HTML elements consider as a box. In CSS, the term box-model is used for layout or design. In CSS box-model is essentially a box that wraps around every HTML element. It consists as a margin, border, padding and a content.

content
The content of the box. Where the text and images appear.
padding
It clears an area around the content. Padding is transparent.
border
border goes around the padding and content
margin
Clears an area outside the border. Margin is transparent.
outline
An outline is a line drawn outside the elements border
outline-offset
It clears an area between outline and border of an element

display property

Display property is most important CSS property for controlling layout.
Every HTML element has a default display value depending on what type of element it is.The default display value for most elements is block or inline.

Block-Level elements

Block-level elements always starts with a new line and takes up the full width available (strecthes out to the left and right as far as it can).

Block-level elements are:
div element, h1 to h6, form, header, footer and section

inline elements

inline elements does not start with new line and takes up as much width as necessary.

inline elements are
span, a, img

display:none;

display:none property is used with JavaScript to hide and show elements without deleting and recreating them

CSS Max-width

Setting the width of the block level element will prevent it from streching out to the edges od its container. Then, we can set the margin to auto, to horizontally center the element within its container.

Using max-width instead, in this situation will improve the browser handling smaller windows.

CSS Position

The position property specifies the type of positioning method used for an element(static, relative, fixed, absolute and sticky)

position:static

HTML elements are positioned static by default.
An element is positioned static, is not positioned in any special way, it is always positioned according to the normal flow of the page.

positioned static

position:relative

An elemnet position:relative; is positioned relative to its normal position.
Setting top, left, right and botton properties set relatively positioned will cause it to be adjusted away from its normal posiiton. Other content will not be adjusted to fit it any gap left by the element.

position:relative;

position:fixed

An element positioned:fixed; it always stays in the same place even if the page scrolls.

position:fixed;
