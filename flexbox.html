CSS Flexbox

CSS Flexbox layout module:

Before flexbox layout module, there were 4 layout modes

  • Block--for sections in a web page
  • inline--for text
  • table--for two-dimensional table data
  • positioned--for explicit position of an element

The flexble box layout module, make it easier to define flexible responsive layout structure without using float or positioned.
