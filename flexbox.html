CSS Flexbox

CSS Flexbox layout module:

Before flexbox layout module, there were 4 layout modes

  • Block--for sections in a web page
  • inline--for text
  • table--for two-dimensional table data
  • positioned--for explicit position of an element

CSS flexbox Vs grid

  • Flexbox:items in a single dimension -- in a row or column
    Grid: items in two dimensions--rows or columns
  • Flexbox--Content based(Content first)
    Grid--Layout based(Layout first)

justify-content Vs align-items Vs align-self

  • justify-content: control alignment of all items on the main-axis.
  • align-items: control alignment of all items on the cross axis
  • align-self: control alignment of an individual flex item on the cross axis

The flexble box layout module, make it easier to define flexible responsive layout structure without using float or positioned.
